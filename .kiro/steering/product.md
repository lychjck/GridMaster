# 产品概述

GridMaster（红利低波网格分析工具）是一个量化网格交易分析工具，支持行情可视化、网格参数配置和回测模拟，覆盖多个资产类别。

## 支持市场
- A股（通过 mootdx / 东方财富）
- 港股
- 加密货币（币安 USDT 交易对）
- 黄金（XAU，通过新浪接口）

## 核心功能
- 分钟级 K 线可视化（1分钟、5分钟）+ 日 K 线，基于 ECharts
- 网格交易回测模拟，含高级指标（夏普比率、年化收益率、最大回撤、胜率）
- 批量参数扫描（不同网格步长的收益对比）
- 交易时段内通过 WebSocket 实时推送行情更新
- 后台按市场分别定时刷新（A股、港股、币安各自独立周期）
- 标的管理（添加/删除/同步）
- 多主题 UI，用户偏好通过 localStorage 持久化

## 语言约定
- UI 文案和代码注释以中文（简体）为主
- 代码标识符和 API 路径使用英文
